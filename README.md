# 부산 지역 맛집 리스트 및 교통편 통합 웹서비스

## 프로젝트 소개
부산 지역 음식점 정보를 통합 제공하는 React + Spring Boot 기반 웹서비스 프로젝트입니다.

공공데이터 API와 네이버 지도 API를 활용하여 음식점 목록, 상세정보, 리뷰, 평점, 교통편, 주차 정보를 제공합니다.

---

## 개발 환경

### Frontend
- React 19
- Vite 8
- Axios
- React Router DOM

### Backend
- Spring Boot 3.2.5
- Java 17
- MyBatis 3.0.3
- Maven

### Database
- MySQL 8.0.34

### 개발 도구
- Eclipse IDE 2024-06
- VS Code 1.109.0
- Git 2.53.0

---

## 주요 기능

- 음식점 목록 조회
- 음식점 검색
- 지역 필터
- 음식점 상세 조회
- 리뷰 작성 및 삭제
- 평균 평점 자동 계산
- 네이버 지도 연동
- 지하철/버스/주차장 교통편 제공

---

## 사용 API

- 부산맛집 공공데이터 API
- 부산 BIMS 버스정보 API
- 부산 공영주차장 API
- 네이버 지도 API
